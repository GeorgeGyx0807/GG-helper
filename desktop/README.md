# Poppy Desktop

Poppy 的 macOS 桌面壳，使用 Tauri 2、React 和 TypeScript，并在启动时管理打包后的 Python Gateway sidecar。

完整的使用、构建和验证说明见仓库中的 `docs/poppy-desktop.md`。

## 开发

```bash
npm install
npm run tauri dev
```
